# Smart Calculator

## Requirements

- python  
- sympy  
- pint  
- Github  

## About

Basically this calculator is smart enough to perform any kind of calculations like summation, addition , multiplications etc. along with some kind of type conversion means conversion of degree to celsius and some weight and unit convertors. also it can store the values and one can use it later on to perform the operations further.

## Features  

- Performs Basic Mathematical operations
- Can also handle user entered complex expressions
- Can also do the unit conversions  
      -- Temperature Conversion  
      -- Weight conversion  
      -- Length Conversion
- can store memory
- can recall that memory
- can remove/clean the memory
